# 🌌 Neon-Shift

**Neon-Shift**는 Three.js로 제작된 스타일리시한 3D 무한 러너 게임입니다. 네온 빛으로 가득한 사이버 공간에서 장애물을 피하며 한계에 도전하세요!

## 🚀 주요 특징
- **5개 레인 시스템:** 더 넓은 시야와 전략적인 움직임이 필요합니다.
- **5단계 난이도:** Beginner(1단계)부터 불가능에 가까운 HELL(5단계)까지 지원합니다.
- **지능형 장애물 생성:** "억까" 상황을 방지하면서도 긴장감을 유지하는 스마트한 패턴 생성 로직.
- **멀티 플랫폼 지원:** 데스크탑(키보드)과 안드로이드(터치) 모두에서 완벽하게 작동합니다.
- **자동 빌드 시스템:** GitHub Actions를 통해 푸시할 때마다 최신 APK가 생성됩니다.

## 🎮 조작 방법
### 데스크탑 (Desktop)
- **A / Left Arrow:** 왼쪽으로 이동
- **D / Right Arrow:** 오른쪽으로 이동

### 모바일 (Android)
- **화면 왼쪽 터치:** 왼쪽으로 이동
- **화면 오른쪽 터치:** 오른쪽으로 이동

## 🛠 기술 스택
- **Engine:** [Three.js](https://threejs.org/) (WebGL)
- **Native Bridge:** [Capacitor](https://capacitorjs.com/) (Android)
- **CI/CD:** GitHub Actions (Auto APK Build)
- **Language:** JavaScript, CSS3, HTML5

## 📦 설치 및 빌드
1. GitHub Actions 탭에서 최신 `app-debug.apk`를 다운로드합니다.
2. 안드로이드 기기에 설치하여 즐기세요! (출처를 알 수 없는 앱 설치 허용 필요)
